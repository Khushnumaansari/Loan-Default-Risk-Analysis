# Loan Default Risk Analysis

## End-to-End Data Analytics Project using Python, SQL, Power BI & Machine Learning

This project is an end-to-end **Loan Default Risk Analysis** case study built using **Python, SQL, Power BI, and Machine Learning**.

The goal of this project is to analyze borrower behavior, identify key default-risk factors, predict loan default probability, and build a business-ready dashboard that can support better lending and portfolio risk decisions.

---

## Project Overview

Loan defaults can create major financial losses for banks, NBFCs, and lending companies. If high-risk borrowers are not identified early, lenders may approve loans that are more likely to default.

This project converts raw loan application data into a complete analytics solution by combining:

- Python-based data cleaning and feature engineering
- Exploratory Data Analysis
- Machine Learning default prediction
- SQL-based business analysis
- Power BI dashboarding
- Executive-level business recommendations

The project demonstrates how data can support smarter credit-risk management and better lending decisions.

---

## Business Problem

A lending company wants to understand why customers default and how risky borrowers can be identified before loan approval.

The business needs answers to questions such as:

- What is the overall loan default rate?
- Which borrower groups are more likely to default?
- How do credit score, DTI ratio, income, late payments, and bankruptcies affect default risk?
- Which loans should be flagged for additional review?
- What is the financial exposure caused by high-risk borrowers?
- How can a dashboard help executives monitor loan portfolio risk?

---

## Project Objectives

The main objectives of this project are:

1. Clean and prepare raw loan data for analysis.
2. Engineer risk-based financial and borrower features.
3. Perform EDA to identify default patterns.
4. Build a basic Machine Learning model to predict loan default probability.
5. Export a clean prediction-ready dataset.
6. Use SQL to generate business KPIs and risk insights.
7. Build a Power BI dashboard for executive decision-making.
8. Provide actionable recommendations for credit-risk management.
